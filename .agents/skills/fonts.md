# Font Formatting Rules

Use these typography rules whenever generating or updating UI text styles:

1. Headers must always use a sans serif font.
2. All other text must always use a serif font.

Applies to:
- Page titles and section headings (`h1`-`h6`): sans serif
- Body copy, labels, helper text, buttons, inputs, and list text: serif

If a style guide already exists, enforce these rules by selecting compatible sans serif and serif families from that guide.
